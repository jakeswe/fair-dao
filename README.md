Deploy contract 
```shell
npx hardhat run ./scripts/deploy.js --network localhost
```

Deploy contract with test organization
```shell
npx hardhat run ./scripts/deploynodes.js --network localhost
```

Run tests
```shell
npx hardhat test
```

Start blockchain
```shell
npx hardhat node
```
Run react
```shell
npm run dev
```
